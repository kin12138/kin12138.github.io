    cpp 对字符串的处理

    find() funciton

    • 作用: 用于查找某个子字符串(或字符)首次出现的位置。
    • 返回值：查找成功返回该子字符串的起始索引，查找失败返回 std::string::npos
    • str 是要查找的子字符串
    • ch 是要查找的字符
    • pos 是查找起始位置，默认从 0 开始。

    replace() function

    • 作用: 用于替换字符串中的一部分内容。可以替换字符、字符串、用迭代器指定的字符范围。

    • 语法:

    • pos: 要替换的起始位置

    • len: 要替换的字符或子字符串的长度
    • str: 用于替换的字符串
    • subpos, sublen: 如果用于替换的字符串是另一个字符串的一部分，可以指定字符串的起始位置和长度
